# coffee2closure

Fix CoffeeScript compiled output for Google Closure Compiler.

## Installation
   ```javascript
   npm install
   ```

## Tests
   ```javascript
   npm test
   ```

## Development Stack
   ```javascript
   node node_modules/.bin/grunt
   ```
   or
   ```javascript
   grunt
   ```
